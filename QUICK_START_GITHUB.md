## Troubleshooting

### "I don't see 'Publish Branch' button"

**Try this:**
1. Click the `...` (three dots) in the Source Control panel
2. Look for "Publish Branch" or "Publish to GitHub" in the menu
3. If still not there, you're probably already published! Check bottom-left for the sync icon

### "Authentication failed" or "Login error"

**Solution:**
1. Sign in to GitHub in your web browser first: https://github.com
2. Then try publishing again in VS Code
3. Make sure you click "Allow" when prompted

### "Repository name already exists"

**This means:**
- You may already have a repository with this name on GitHub
- Choose a different name, like: `ims-weather-automation-2025`
- Or delete the old repository on GitHub first (if you don't need it)

### "I got an error I don't understand"

**Fallback option - Manual method:**
1. Open **[GITHUB_SETUP.md](GITHUB_SETUP.md)**
2. Scroll to "Method 2: Manual Method"
3. Follow those step-by-step instructions
